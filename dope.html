3D Карточка с Вращением


Frontend · CSS, Анимация · Ноябрь 16, 2022

Создайте двустороннюю карточку, которая вращается при наведении курсора

1. Установите backface-visibility: none для обратной стороны.

2. Установите rotateY() со значением -180deg для обратной стороны, а для лицевой установите 0deg.

3. При наведении курсора установите rotateY() со значением 180deg для лицевой стороны, а для обратной установите 0deg.

4. Установите значение perspective для родителя, создав эффект поворота.

HTML


<div class="card">
 <div class="card-side front">
   <div>Front Side</div>
 </div>
  <div class="card-side back">
   <div>Back Side</div>
  </div>
</div>

CSS


.card {
     perspective: 150rem;
     position: relative;
     height: 40rem;
     max-width: 400px;
     margin: 2rem;
     box-shadow: none;
     background: none;
}
.card-side {
     height: 35rem;
     border-radius: 15px;
     transition:all 0.8s ease;
     backface-visibility: hidden;
     position: absolute;
     top: 0;
     left: 0;
     width: 80%;
     padding: 2rem;
     color: white;
}
.card-side.back {
     transform: rotateY(-180deg);
     background-color: #4158D0;
     background-image: linear-gradient(43deg, #4158D0 0%,#C850C0 46%, #FFCC70 100%);
}
.card-side.front {
     background-color: #0093E9;
     background-image: linear-gradient(160deg, #0093E9 0%, #80D0C7 100%);
}
.card:hover .card-side.front {
     transform: rotateY(180deg);
}
.card:hover .card-side.back {
     transform: rotateY(0deg);
}
